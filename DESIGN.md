# Project Motivation
This project was inspired by the lectures about accessibility for this class. Any good application should take into consideration the needs of those with disabilities. With Ariabl, developers can paste or upload their frontend code to recieve and review AI-powered corrections

## Contributions
As the sole contributor (Mason DeWitt), I will be developing the project in its entirety. I am going to fully develop the API layer before moving on to the client.

### External Data Sources
This project will use more than a handful of NPM packages to achieve the desired outcome. For example, I have already used the package `htmlparser2` to assist in separating HTML code into individual tags and flagging structural errors. As for more complex external API integration, I am planning on using the Claude API for AI-powered accessibility corrections.
